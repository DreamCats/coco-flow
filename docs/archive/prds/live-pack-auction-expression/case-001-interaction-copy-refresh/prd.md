# PRD

## 需求名称

竞拍爽感实验下优化讲解卡引导文案

## 背景

- 当前竞拍讲解卡在预热态和无人出价场景下，文案对“赶快出第一口价”的引导不够直接
- 实验组希望把讲解卡文案改得更轻、更短、更偏行动引导

## 目标

- 命中实验时，让用户更快理解当前价格是起拍价
- 命中实验时，让用户更清楚现在还没人出价，自己可以成为第一位出价者
- regular auction 和 surprise set 保持各自玩法语义，但整体表达风格一致

## 范围

- 仅竞拍讲解卡
- 仅文案表达变化
- 覆盖 regular auction 和 surprise set
- 覆盖预热态和无人出价的开场态

## 非目标

- 不改按钮文案
- 不改价格本身
- 不改购物袋
- 不新增前端字段

## 用户故事

作为直播间里的潜在竞拍用户，我希望在看到竞拍卡时，能立刻知道现在展示的是起拍价，并且知道当前还没人出价，这样我更容易决定是否参与第一口竞拍。

## 验收标准

1. 命中实验时，regular auction 在预热态展示更直接的 `Starting price` 引导文案。
2. 命中实验时，regular auction 在竞拍开始但还没人出价时，文案明确表达“从起拍价开始，并鼓励用户成为第一位出价者”。
3. 命中实验时，surprise set 在预热态和首拍态也切到同一套爽感表达风格，但保留 surprise 玩法语义。
4. 未命中实验时，regular auction 和 surprise set 保持现有线上文案不变。
