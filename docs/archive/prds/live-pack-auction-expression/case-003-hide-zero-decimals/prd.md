# PRD

## 需求名称

实验组竞拍讲解卡整数金额隐藏尾部 `.00`

## 背景

- 当前竞拍讲解卡在展示整数金额时会带上 `.00`，视觉上偏重，价格信息不够干净
- 实验组希望在不改变金额含义的前提下，让整数价格展示更简洁

## 目标

- 命中实验时，讲解卡里的整数金额展示去掉尾部 `.00`
- 非整数金额保持原有格式
- 改动只影响讲解卡，不扩大到其他价格链路

## 范围

- 仅竞拍讲解卡
- 覆盖 regular auction 和 surprise set
- 覆盖讲解卡里对用户可见的竞拍价格项

## 非目标

- 不改真实金额值
- 不改非整数金额展示
- 不改购物袋价格展示
- 不改底层通用价格格式化规则

## 用户故事

作为浏览竞拍讲解卡的用户，我希望价格看起来更清爽。如果金额本身就是整数，我不希望再额外看到 `.00` 这种没有信息增量的尾部。

## 验收标准

1. 命中实验时，整数金额例如 `12.00` 展示为 `12`。
2. 命中实验时，非整数金额例如 `12.50` 仍保持原有格式。
3. regular auction 和 surprise set 的讲解卡价格展示口径一致。
4. 未命中实验时，讲解卡价格展示保持现有线上逻辑不变。
5. 购物袋等非讲解卡价格链路不受影响。
