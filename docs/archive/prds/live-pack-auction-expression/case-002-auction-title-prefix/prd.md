# PRD

## 需求名称

实验组竞拍讲解卡标题增加 `Auction` 标识

## 背景

- regular auction 在讲解卡里直接使用商品标题时，用户对“这是竞拍商品”这一点的感知不够强
- 实验组希望通过更轻量的标题前缀，提升讲解卡的竞拍辨识度

## 目标

- 命中实验时，让 regular auction 标题更容易被识别为竞拍卡
- 不改商品标题主体内容，只在前面增加清晰、简短的竞拍标识
- 标题变化应保持本地化能力

## 范围

- 仅 regular auction 讲解卡标题
- 仅实验命中场景
- 仅标题前缀表达变化

## 非目标

- 不改价格
- 不改按钮
- 不改 surprise set 和 temporary listing
- 不改购物袋标题

## 用户故事

作为正在浏览直播间商品的用户，我希望一眼就能从标题上看出这是一张竞拍卡，而不是普通商品卡，这样我能更快理解当前卡片的玩法。

## 验收标准

1. 命中实验时，regular auction 标题前增加本地化的 `Auction` 标识。
2. 标题前缀与原标题之间有稳定分隔，整体可读性不受影响。
3. 如果本地化标识取值异常为空，则回退为原标题，不出现空前缀或异常连接符。
4. 未命中实验时，标题保持现有线上逻辑不变。
