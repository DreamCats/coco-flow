# PRD

## 需求名称

多 SKU 竞拍讲解卡标题补充 SKU 名前缀

## 背景

当前多规格商品参与竞拍时，讲解卡标题通常只展示商品主标题，不会明确告诉用户当前参拍的是哪个规格。

对于颜色、容量、套装这类差异明显的商品，用户仅靠标题很难判断当前正在竞拍的具体规格。

## 目标

当竞拍商品存在多个规格，且当前竞拍规格有明确规格名时，讲解卡标题改为：

`#<sku_name> <商品标题>`

## 范围

- 普通竞拍讲解卡
- 多规格商品
- 不同讲解卡返回路径的标题口径需要一致

## 非目标

- 不改商品主标题来源
- 不改图片、schema、价格
- 不处理单 SKU 商品

## 用户故事

作为直播间用户，我希望在竞拍讲解卡上直接看到当前参拍规格，这样我不用额外点进 PDP 再确认是哪个 SKU。

## 验收标准

1. 当商品有多个规格，且当前竞拍规格存在规格名时，标题展示为 `#规格名 商品标题`
2. 当商品只有一个规格，或规格名为空时，维持原商品标题
3. 已有支持规格前缀的竞拍类型不允许回退
4. 同一竞拍商品在不同讲解卡返回路径下，标题规则一致
