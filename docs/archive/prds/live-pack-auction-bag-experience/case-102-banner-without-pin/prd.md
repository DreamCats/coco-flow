# PRD

## 需求名称

无 Pin 竞拍卡时购物袋 banner 仍可展示

## 背景

- 当前购物袋 banner 过度依赖 pin 竞拍卡，导致某些没有 pin 的场景下，即使列表里已有竞拍商品，也无法展示 banner
- 产品希望只要当前页存在可展示的竞拍商品，banner 就不要因为“没有 pin”而缺席

## 目标

- 当购物袋里存在竞拍商品且 banner 条件成立时，即使当前没有 pin 竞拍卡，也能正常展示 banner
- 有 pin 时仍优先使用 pin 对应的竞拍商品
- 无 pin 时回退到当前列表中的第一个竞拍商品

## 范围

- 仅购物袋 `AuctionInfoBanner`
- 仅 banner 的锚定对象选择逻辑
- 不改 banner 文案和展示资格判断

## 非目标

- 不改默认 tab
- 不改竞拍商品排序
- 不改 banner 文案内容
- 不改无竞拍商品时的返回行为

## 用户故事

作为正在看购物袋的用户，只要当前页确实有竞拍商品，我希望顶部竞拍 banner 也能正常出现，而不是因为当前没有 pin 卡就完全看不到入口。

## 验收标准

1. 有 pin 竞拍商品时，banner 继续优先绑定 pin 对应商品。
2. 无 pin 但当前页有竞拍商品时，banner 回退绑定当前列表中的第一个竞拍商品。
3. 当前页没有竞拍商品时，不因为这次改动额外生成 banner。
4. banner 的文案和展示资格判断保持现有逻辑不变。
