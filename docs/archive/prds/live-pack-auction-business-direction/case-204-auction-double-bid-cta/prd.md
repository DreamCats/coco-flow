# PRD

## 需求名称

竞拍卡支持快捷出价和自定义出价双 CTA

## 背景

- 竞拍用户既有“按下一口价快速出价”的需求，也有“输入更高心理价位”的需求
- 单一 CTA 容易让用户误以为只能按系统推荐价格出价
- 自定义出价入口如果藏得太深，会降低高意愿用户的出价效率

## 目标

- 命中实验时，竞拍卡展示快捷出价和自定义出价两个入口
- regular auction 和 surprise set 保持一致的双入口心智
- 预热态不展示会误导用户出价的按钮文案
- 未命中实验时保持现有单入口行为

## 范围

- 覆盖直播间竞拍讲解卡
- 覆盖购物袋竞拍卡
- 覆盖 regular auction 和 surprise set
- 按现有 bid panel 和 customize panel 结构下发

## 非目标

- 不改竞拍价格计算
- 不改自定义出价校验规则
- 不改前端按钮布局细节
- 不新增竞拍 RPC
- 不改变支付或协议签署逻辑

## 用户故事

作为参与竞拍的用户，我希望能直接选择快速出价或自定义出价，这样我既可以低成本参与，也可以在意愿强时更快给出自己的最高价。

## 验收标准

1. 命中实验时，竞拍卡下发双 CTA 配置。
2. 快捷出价 CTA 使用下一口价或当前可出价信息。
3. 自定义出价 CTA 能打开自定义出价面板。
4. 预热态不展示可点击出价误导文案。
5. 未命中实验时保持现有单 CTA 结构和文案。
