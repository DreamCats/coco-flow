# PRD

## 需求名称

竞拍卡展示商品营销标签

## 背景

- 竞拍商品同样可能具备免运费、时效、权益或优惠等商品营销信息
- 如果竞拍卡和购物袋竞拍列表不展示这些标签，用户会低估竞拍商品权益
- regular auction 和 surprise set 都需要保持基础权益信息可见，但 surprise set 应避免展示与奖池无关或误导性的标签

## 目标

- 命中实验时，竞拍讲解卡展示商品营销标签
- 购物袋竞拍商品卡也展示同一口径的营销标签
- surprise set 详情页奖品只展示适合奖品维度的标签，如运费和履约类标签
- 未命中实验时保持现有竞拍卡样式

## 范围

- 覆盖 regular auction pop card
- 覆盖 surprise set pop card 和 surprise set detail available item
- 覆盖购物袋竞拍列表和 refresh
- 标签来源复用商品模型中的 promotion labels

## 非目标

- 不新增营销标签计算逻辑
- 不改商品价格或券计算
- 不改变标签文案来源
- 不改非竞拍商品卡
- 不改前端标签组件样式

## 用户故事

作为直播间用户，我希望竞拍商品也能展示免运费、履约等关键信息，这样我能更准确地判断竞拍商品的真实吸引力。

## 验收标准

1. 命中实验时，regular auction 讲解卡展示商品营销标签。
2. 命中实验时，购物袋竞拍商品卡展示商品营销标签。
3. surprise set 详情页奖品只展示允许的运费/履约类标签。
4. 未命中实验时，竞拍卡和购物袋竞拍卡保持现有标签展示。
5. 标签数据缺失时不展示空占位，不影响竞拍卡其它字段。
