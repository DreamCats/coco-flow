# PRD

## 需求名称

Surprise Set 奖品支持直达商品详情页

## 背景

- surprise set 详情页会展示可获得奖品列表
- 用户看到某个奖品后，可能希望进一步查看商品详情、规格和履约信息
- 当前若只展示奖品信息但缺少可点击 schema，用户需要回到购物袋或直播间其它入口查找商品

## 目标

- surprise set 详情页中的可获得奖品支持跳转商品详情页
- 跳转时带上该奖品对应的 SKU 规格信息
- 能通过实验开关控制是否下发奖品 schema
- 未命中实验或 schema 拼装失败时，保持现有不可跳转或原 schema 行为

## 范围

- 仅 surprise set 详情页奖品列表
- 覆盖可获得奖品，不要求已售出奖品可跳转
- 仅在后端能拿到商品 PDP schema 和 SKU 属性时下发
- 需要支持 regular PDP schema 追加 `checked_spec_ids`

## 非目标

- 不改 surprise set 玩法规则
- 不改库存扣减逻辑
- 不改出价流程
- 不改商品详情页承接能力
- 不新增奖品排序策略

## 用户故事

作为参与 surprise set 的用户，我希望能点击某个可能获得的奖品查看商品详情，这样我能更清楚地判断是否值得参与竞拍。

## 验收标准

1. 命中实验时，可获得奖品项下发可点击 schema。
2. schema 跳转到对应商品详情页，并默认选中该 surprise set item 的 SKU。
3. 未命中实验时，不新增奖品 schema。
4. 商品模型缺少 PDP schema 或 SKU 信息时，不下发异常 schema。
5. 已售出奖品列表不因本需求改变展示和跳转行为。
