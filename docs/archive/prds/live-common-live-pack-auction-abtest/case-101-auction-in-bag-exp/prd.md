# PRD

## 需求名称

增加竞拍进购物袋实验开关并接通购物袋 auction 链路

## 背景

竞拍商品进入购物袋属于新能力，当前希望先通过实验对小流量开放，再逐步扩大范围。

为了避免实验外用户误看到未准备好的竞拍购物袋体验，需要在功能入口和结果返回上都做好隔离。

## 目标

新增“竞拍进购物袋”实验开关。命中实验时，购物袋才展示和返回竞拍相关能力；未命中时，购物袋保持原有普通商品路径。

## 改动点

1. 增加实验开关
2. 在购物袋主入口上按实验命中与否控制竞拍能力是否生效
3. 在相关辅助返回里同步尊重该开关，避免实验外混入竞拍数据

## 范围

- 购物袋 auction list / assemble / refresh
- room lite 里的 auction 相关输出
- user right 中竞拍品 ID 合并逻辑

## 非目标

- 不调整竞拍卡片样式
- 不修改 auction banner 内容
- 不做新的 auction 表达层文案设计

## 验收标准

1. 实验未命中时，购物袋不展示竞拍能力
2. 实验命中时，购物袋可正常展示竞拍相关内容
3. 实验外用户不会收到竞拍商品、竞拍 banner 等结果
4. 与购物袋相关的其他辅助返回也要和实验口径一致
