# PRD

## 需求名称

增加竞拍营销标签实验，控制讲解卡与购物袋的 promotion label 表达

## 背景

竞拍场景希望测试更强的促销表达，让用户更容易感知当前竞拍商品的促销信息。

这类表达希望先按实验小流量验证效果，而不是直接全量开放。

## 目标

新增竞拍营销标签实验。命中实验时，讲解卡和购物袋都展示更强的促销标签或促销文案；未命中实验时，保持当前展示。

## 改动点

1. 增加实验开关
2. 普通竞拍讲解卡按实验控制营销标签展示
3. 购物袋和惊喜盲盒竞拍也按同样口径控制促销表达

## 范围

- popcard regular auction
- bag regular auction
- surprise set promotion text

## 非目标

- 不调整 placement label
- 不修改价格和按钮
- 不新增 TCC 文案结构

## 验收标准

1. 实验未命中时，不展示新增的竞拍促销表达
2. 实验命中时，讲解卡和购物袋都能展示竞拍促销标签
3. 惊喜盲盒竞拍的促销文案也受该实验控制
4. 讲解卡和购物袋不允许出现一边展示、一边不展示的口径漂移
