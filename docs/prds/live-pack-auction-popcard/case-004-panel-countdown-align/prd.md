# PRD

## 需求名称

旧兼容链路竞拍面板标题与倒计时标题对齐新链路

## 背景

当前竞拍讲解卡的出价面板和自定义出价面板，在不同返回路径下存在标题不一致的问题：

- 有的路径会展示倒计时标题，有的不会
- 自定义出价面板会错误复用普通出价面板标题

这会让用户在不同入口下看到的竞拍面板信息不一致。

## 目标

把竞拍面板的标题和倒计时标题表达统一，避免不同返回路径下出现 UI 不一致。

## 改动点

1. 普通出价面板补充倒计时标题
2. 自定义出价面板补充倒计时标题
3. 自定义出价面板使用自己的标题配置，不再复用普通出价面板标题

## 范围

- 仅普通竞拍讲解卡
- 仅修正面板标题和倒计时标题表达
- 不改购物袋

## 非目标

- 不改倒计时触发时机
- 不改 TCC 字段定义
- 不改按钮文案和价格展示

## 验收标准

1. 普通出价面板在需要时能展示倒计时标题
2. 自定义出价面板在需要时也能展示倒计时标题
3. 自定义出价面板标题与普通出价面板标题解耦
4. 已经正确展示的场景保持不变
