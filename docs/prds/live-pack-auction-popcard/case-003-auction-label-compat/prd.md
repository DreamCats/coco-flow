# PRD

## 需求名称

旧兼容链路 regular auction 也下发拍卖标签

## 背景

当前普通竞拍讲解卡在部分返回路径下会展示左上角的“拍卖”标签，但在另一些返回路径下该标签会缺失。

这会导致用户在不同入口或设备上看到的竞拍讲解卡样式不一致。

## 目标

让普通竞拍讲解卡在所有返回路径下都稳定展示统一的拍卖标签，保证样式一致。

## 范围

- 仅普通竞拍讲解卡
- 仅补齐缺失标签的返回路径
- 不改标签文案来源

## 非目标

- 不改 SurpriseSet 标签
- 不改 placement label / promotion label
- 不改标签文案来源

## 用户故事

作为直播间用户，我希望无论命中新链路还是旧兼容链路，竞拍讲解卡左上角都能稳定展示“拍卖”标签，不出现一会儿有一会儿没有的情况。

## 验收标准

1. 普通竞拍讲解卡在所有返回路径下都能稳定展示拍卖标签
2. 标签文案、颜色、图标保持一致
3. 当前已经正确展示标签的场景不受影响
4. 不影响其他已有标签的展示
