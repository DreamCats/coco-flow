# PRD

## 需求名称

竞拍讲解卡新增扩展展示字段

## 背景

前端希望在竞拍讲解卡上展示一项新的扩展信息，用来提升用户对当前竞拍状态或玩法的理解。

当前该字段在 API 层没有，也没有稳定从下游能力中透出，因此前端无法直接消费。

## 目标

为竞拍讲解卡新增一个可稳定下发的扩展展示字段，保证前端可以直接读取并展示。

## 范围

- 仅竞拍讲解卡相关接口
- 涉及服务间字段透传与 API 字段新增
- 不改变现有竞拍核心逻辑

## 非目标

- 不改动竞拍排序、出价逻辑、支付逻辑
- 不改动客户端展示样式
- 不新增新的业务规则判断

## 验收标准

1. 竞拍讲解卡 API 中新增扩展展示字段
2. 字段来源稳定，可由下游服务透传到 API
3. 不影响现有讲解卡主体内容和兼容行为
